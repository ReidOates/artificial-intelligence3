## 🌟 Fitur Utama

- **🌍 Default Dataset**: Langsung siap digunakan karena memiliki *dataset* bawaan (Apple Inc. - AAPL).
- **📂 Upload Data Kustom**: Memungkinkan pengguna mengunggah *dataset* saham mereka sendiri (format CSV) untuk dianalisis.
- **🎯 Prediksi Interaktif**: Pengguna bisa memasukkan harga pembukaan (Open) secara manual, lalu model akan memprediksi harga penutupan (Close).
- **📊 Visualisasi Data**: Menampilkan plot grafik regresi linear yang interaktif dan informatif menggunakan Matplotlib & Seaborn.
- **📈 Metrik Evaluasi Model**: Menyediakan nilai MAE (Mean Absolute Error), MSE (Mean Squared Error), serta R-Squared (R²) secara transparan untuk menilai performa kecerdasan buatan.

## 🛠️ Tech Stack 

- **Backend**: Python 3, Flask

- **Data Processing**: Pandas, NumPy
- **Data Visualization**: Matplotlib, Seaborn
- **Frontend**: HTML5, CSS3, Bootstrap
